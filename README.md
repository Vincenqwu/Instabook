## Deployment

Using Linode & Cloudflare

### Backend

1. setup correct parameters in .env
1. ```npm install```
1. ```npm prisma generate```
1. ```npm prisma db push```
1. ```npm start```
   
### Frontend

1. setup correct parameters in .env
1. ```npm install```
1. ```npm start```

### Database

Using Mongodb Atlas
